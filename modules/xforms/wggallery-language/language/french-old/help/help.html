Aide : wgGallery

Description

wgGallery peut être utilisé pour présenter vos images dans des albums au sein de votre XOOPS

Installation/désinstallation

Aucune mesure particulière n'est nécessaire, suivez le processus d'installation standard et extrayez le dossier wggallery dans le répertoire ../modules. Installez le module via Admin -> Module système -> Modules.

Des instructions détaillées sur l'installation des modules sont disponibles dans le Manuel d'exploitation XOOPS

Caractéristiques

Le module wgGallery continue de s'étendre, pour arriver aux conditions de création de modules, de plus en plus sophistiqués.
Pour cette raison, j'invite tous les développeurs à signaler et envoyer à github tout changement ou ajout à ce module, afin que nous puissions contribuer conjointement au développement
