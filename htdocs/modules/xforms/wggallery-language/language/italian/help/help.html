Aiuto: wgGallery

Descrizione

Il modulo wgGallery può essere utilizzato per presentare le immagini negli album internamente al vostro XOOPS

Installa/disinstalla

Non sono necessarie misure speciali, seguire la procedura di installazione standard ed estrarre la cartella wggallery nella directory ../modules. Installa il modulo tramite Admin -> Modulo di sistema -> Moduli.
Istruzioni dettagliate sulla installazione dei moduli sono disponibili nel Manuale operativo di XOOPS

Caratteristiche

Il modulo wgGallery continua ad espandersi, per arrivare alla possibilità di creare moduli sempre più sofisticati.
Per questo motivo, invito tutti gli sviluppatori a segnalare e inviare a github ogni modifica o aggiunta a questo modulo, in modo che possiamo contribuire congiuntamente allo sviluppo
