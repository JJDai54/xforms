Help: wgGallery

Description

Das wgGallery-Modul kann zur Präsentation Ihrer Bilder in Alben auf Ihrem XOOPS verwendet werden.

(de-)Installieren

Es sind keine speziellen Maßnahmen erforderlich, führen Sie den Standardinstallationsprozess aus:
- Extrahieren Sie den Ordner wgSiteNotice in das Module-Verzeichnis
- Installieren Sie das Modul über Administration -> Module
Wenn Sie zustätzliche Informationen benötigen, lesen Sie die Installationshinweise unter XOOPS Operations Manual.

Funktionen

Das wgGallery-Modul wird noch weiter entwickelt, um so ein umfassendes Modul für Bildergallerien zu werden.
Aus diesem Grund sind alle Entwickler eingeladen, Verbesserungs- oder Änderungsvorschläge auf github einzusenden.
